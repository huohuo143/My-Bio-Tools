# My Bio Tools 1.9.7（Build 26）

本版在水稻基因分析的“大模型增强解读”中新增 API-only 接入：

- DeepSeek API
- 豆包 API（火山方舟）
- 智谱 GLM API
- 通义千问 API（阿里云百炼）
- ChatAnywhere API
- OpenAI / 自定义兼容 API

API 地址和模型名称可编辑。每个提供方使用独立 API Key 输入框；API Key 只保留在当前 APP 会话内存，退出后需要重新输入，不写入偏好文件、报告、日志或发布物。连接测试只发送固定文本；正式解读仅发送去标识化结构化证据，不发送原始序列、图片、样本名、密码、令牌、密钥或本地路径。云端调用失败时自动回退离线科研规则。

同时包含 v1.9.5 的账号授权期限/访客基础工具支持，以及 v1.9.6 的可统计多组学、机制证据链和 AI 深度机制报告增强。

验证结果：多供应商测试 6/6、报告解读 5/5、核心功能 36/36、页面交互流程 6/6、Swift 测试 9/9、授权服务测试 19/19 均通过；冻结后端、源码完整性、DMG 完整性及安装后运行健康检查均通过。

本次仅发布经过本机验证的 macOS Apple Silicon 安装包及 SHA-256；未发布 Windows 二进制。安装包为 ad-hoc 签名，未进行 Apple notarization。
